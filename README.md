# Vania Kristianto | IT & Creative Portfolio

A modern, responsive portfolio website showcasing my IT expertise and video editing work.

## Features

- **Responsive Design**: Mobile-first approach with smooth breakpoints
- **Dark/Light Theme**: Toggle between light and dark mode with persistent storage
- **Interactive Gallery**: Filterable portfolio sections for IT and Video Editing
- **Video Embeds**: Direct YouTube, Instagram, Google Drive, and Vimeo embeds in modal previews
- **Smooth Animations**: Scroll-based reveal animations and transitions
- **Accessibility**: ARIA labels, semantic HTML, keyboard navigation

## Sections

- **Hero**: Eye-catching intro with CTA buttons
- **About**: Personal introduction with CV download and contact details
- **Resume**: Education and professional summary
- **Portfolio**: 
  - IT Portfolio (Networking, IoT & AI projects)
  - Video Editing Portfolio (Documentary, Motion Graphics, Visual Effects, Animation)
- **Contact**: Links to reach out via WhatsApp or email

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript (ES6+)
- Font Awesome Icons
- Bootstrap Icons

## File Structure

```
├── vania_portfolio_index.html  # Main HTML file
├── styles.css                   # All styling
├── script.js                    # JavaScript logic
├── img/                         # Image assets
└── README.md                    # This file
```

## Usage

Simply open `vania_portfolio_index.html` in a web browser, or host on GitHub Pages:

1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Select branch as source
